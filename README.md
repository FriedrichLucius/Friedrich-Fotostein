Welcome to my GitHub profile. I am an independent researcher focused on non‑invasive brain‑computer interfaces (BCIs) for human memory enhancement. ## 🔬 Current Project: MemLink
- **Non‑invasive closed‑loop BCI** using scalp EEG and focused ultrasound neuromodulation.
- Targets hippocampal replay during sleep to strengthen episodic memory consolidation.
- Open‑source hardware (BOM < $80) and free software.
- **Status:** Prototype design, IRB preparation, seeking funding (applied to EA Funds – Long‑Term Future Fund).
